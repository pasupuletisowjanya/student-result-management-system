# student-result-management-system
Student Result Management System using MS Excel for managing marks and results
A simple academic result management project built using MS Excel.

### Features

* Student record management

* Total marks calculation

* Percentage calculation

* Pass/Fail status determination

* Organized data storage using Excel tables

### Tools Used

* MS Excel

### Project Objective

To automate student result processing and maintain academic records efficiently with reduced manual errors.
